Herança e Especificidade em CSS

Assim como você herda métodos e atributos de objetos, no CSS você herda as regras de um elemento pai. É importante lembrar que nem todas as propriedades serão herdadas por elementos filho.

No caso de dois Seletores Iguais, o que estiver por ultimo em relação ao código CSS irá prevalescer

Se um seletor for mais especifico, a regra mais especifica será aplicada
